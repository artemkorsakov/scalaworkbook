### Inline Constants

–ü—Ä–æ—Å—Ç–µ–π—à–µ–π —Ñ–æ—Ä–º–æ–π –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–∏—è —è–≤–ª—è–µ—Ç—Å—è –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–∏–µ –∫–æ–Ω—Å—Ç–∞–Ω—Ç –≤ –ø—Ä–æ–≥—Ä–∞–º–º—ã:

–ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ –∫–ª—é—á–µ–≤–æ–≥–æ —Å–ª–æ–≤–∞ `inline` –≤ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è—Ö –∑–Ω–∞—á–µ–Ω–∏–π –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç,


–í –ø—Ä–∏–≤–µ–¥–µ–Ω–Ω–æ–º –≤—ã—à–µ –∫–æ–¥–µ —Å—Å—ã–ª–∫–∏ `pi` –∏ `pie` –≤—Å—Ç—Ä–æ–µ–Ω—ã. 
–ó–∞—Ç–µ–º –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –ø—Ä–∏–º–µ–Ω—è–µ—Ç –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏—é –ø–æ–¥ –Ω–∞–∑–≤–∞–Ω–∏–µ–º "—Å–≤–µ—Ä—Ç—ã–≤–∞–Ω–∏–µ –∫–æ–Ω—Å—Ç–∞–Ω—Ç", 
–∫–æ—Ç–æ—Ä–∞—è –≤—ã—á–∏—Å–ª—è–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∏—Ä—É—é—â–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ `pi2` –∏ `pie2` –≤–æ –≤—Ä–µ–º—è –∫–æ–º–ø–∏–ª—è—Ü–∏–∏.

> Inline (Scala 3) vs. final (Scala 2)
> 
> –í Scala 2 –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–ª—Å—è –±—ã –º–æ–¥–∏—Ñ–∏–∫–∞—Ç–æ—Ä `final` –≤ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–∏ –±–µ–∑ –≤–æ–∑–≤—Ä–∞—â–∞–µ–º–æ–≥–æ —Ç–∏–ø–∞:
> 
> ```scala
> final val pi = 3.141592653589793
> final val pie = "ü•ß"
> ```
> 
> –ú–æ–¥–∏—Ñ–∏–∫–∞—Ç–æ—Ä `final` –æ–±–µ—Å–ø–µ—á–∏—Ç, —á—Ç–æ `pi` –∏ `pie` –ø—Ä–∏–º–µ—Ç –ª–∏—Ç–µ—Ä–∞–ª—å–Ω—ã–π —Ç–∏–ø. 
> –ó–∞—Ç–µ–º –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏—è —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–∏—è –∫–æ–Ω—Å—Ç–∞–Ω—Ç –≤ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–µ –º–æ–∂–µ—Ç –≤—ã–ø–æ–ª–Ω–∏—Ç—å –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–∏–µ –¥–ª—è —Ç–∞–∫–∏—Ö –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–π. 
> –û–¥–Ω–∞–∫–æ —ç—Ç–∞ —Ñ–æ—Ä–º–∞ –ø–æ—Å—Ç–æ—è–Ω–Ω–æ–≥–æ —Ä–∞—Å–ø—Ä–æ—Å—Ç—Ä–∞–Ω–µ–Ω–∏—è –Ω–µ –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç—Å—è. 
> Scala 3.0 —Ç–∞–∫–∂–µ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç `final val` - inlining –∫–∞–∫ –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–∏–µ —Å –º–∞–∫—Å–∏–º–∞–ª—å–Ω–æ–π —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ—Å—Ç—å—é –¥–ª—è —Ü–µ–ª–µ–π –º–∏–≥—Ä–∞—Ü–∏–∏.

–í –Ω–∞—Å—Ç–æ—è—â–µ–µ –≤—Ä–µ–º—è —Ç–æ–ª—å–∫–æ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–Ω—ã–µ –≤—ã—Ä–∞–∂–µ–Ω–∏—è –º–æ–≥—É—Ç –ø–æ—è–≤–ª—è—Ç—å—Å—è –≤ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –≤—Å—Ç—Ä–æ–µ–Ω–Ω–æ–≥–æ –∑–Ω–∞—á–µ–Ω–∏—è. 
–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, —Å–ª–µ–¥—É—é—â–∏–π –∫–æ–¥ –Ω–µ–¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª–µ–Ω, —Ö–æ—Ç—è –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –∑–Ω–∞–µ—Ç, 
—á—Ç–æ –ø—Ä–∞–≤–∞—è —á–∞—Å—Ç—å —è–≤–ª—è–µ—Ç—Å—è –ø–æ—Å—Ç–æ—è–Ω–Ω—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º –≤—Ä–µ–º–µ–Ω–∏ –∫–æ–º–ø–∏–ª—è—Ü–∏–∏:

```scala

```

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –ø—Ä–∏ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏–∏ `inline val pi` –¥–æ–±–∞–≤–ª–µ–Ω–∏–µ –º–æ–∂–µ—Ç –±—ã—Ç—å –≤—ã—á–∏—Å–ª–µ–Ω–æ –≤–æ –≤—Ä–µ–º—è –∫–æ–º–ø–∏–ª—è—Ü–∏–∏. 
–≠—Ç–æ —É—Å—Ç—Ä–∞–Ω—è–µ—Ç —É–∫–∞–∑–∞–Ω–Ω—É—é –≤—ã—à–µ –æ—à–∏–±–∫—É –∏ `pi2` –ø–æ–ª—É—á–∞–µ—Ç –ª–∏—Ç–µ—Ä–∞–ª—å–Ω—ã–π —Ç–∏–ø `6.283185307179586d`.


### Inline Methods

–¢–∞–∫–∂–µ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –º–æ–¥–∏—Ñ–∏–∫–∞—Ç–æ—Ä `inline` –¥–ª—è –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –º–µ—Ç–æ–¥–∞, –∫–æ—Ç–æ—Ä—ã–π –¥–æ–ª–∂–µ–Ω –±—ã—Ç—å –≤—Å—Ç—Ä–æ–µ–Ω –≤ —Ç–æ—á–∫—É –≤—ã–∑–æ–≤–∞:

```scala
inline def logged[T](level: Int, message: => String)(inline op: T): T =
  println(s"[$level]Computing $message")
  val res = op
  println(s"[$level]Result of $message: $res")
  res
```

–ö–æ–≥–¥–∞ –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è —Ç–∞–∫–æ–π –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã–π –º–µ—Ç–æ–¥ `logged`, –µ–≥–æ —Ç–µ–ª–æ –±—É–¥–µ—Ç —Ä–∞–∑–≤–µ—Ä–Ω—É—Ç–æ –Ω–∞ –º–µ—Å—Ç–µ –≤—ã–∑–æ–≤–∞ –≤–æ –≤—Ä–µ–º—è –∫–æ–º–ø–∏–ª—è—Ü–∏–∏! 
–¢–æ –µ—Å—Ç—å –≤—ã–∑–æ–≤ `logged` –±—É–¥–µ—Ç –∑–∞–º–µ–Ω–µ–Ω —Ç–µ–ª–æ–º –º–µ—Ç–æ–¥–∞. 
–ü—Ä–µ–¥–æ—Å—Ç–∞–≤–ª–µ–Ω–Ω—ã–µ –∞—Ä–≥—É–º–µ–Ω—Ç—ã —Å—Ç–∞—Ç–∏—á–µ—Å–∫–∏ –∑–∞–º–µ–Ω—è—é—Ç—Å—è –ø–∞—Ä–∞–º–µ—Ç—Ä–∞–º–∏ `logged`, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ. 
–ü–æ—ç—Ç–æ–º—É –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –≤—Å—Ç—Ä–∞–∏–≤–∞–µ—Ç —Å–ª–µ–¥—É—é—â–∏–π –≤—ã–∑–æ–≤

```scala

  computeSomething()
}
```

–∏ –ø–µ—Ä–µ–ø–∏—Å—ã–≤–∞–µ—Ç –µ–≥–æ –Ω–∞:

```scala
val level   = logLevel
def message = getMessage()

println(s"[$level]Computing $message")

println(s"[$level]Result of $message: $res")

```

–í—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –º–µ—Ç–æ–¥—ã –≤—Å–µ–≥–¥–∞ –¥–æ–ª–∂–Ω—ã –ø—Ä–∏–º–µ–Ω—è—Ç—å—Å—è –ø–æ–ª–Ω–æ—Å—Ç—å—é. –ù–∞–ø—Ä–∏–º–µ—Ä, –≤—ã–∑–æ–≤

```

```

–±—É–¥–µ—Ç –Ω–µ–ø—Ä–∞–≤–∏–ª—å–Ω–æ —Å—Ñ–æ—Ä–º–∏—Ä–æ–≤–∞–Ω, –∏ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –±—É–¥–µ—Ç –∂–∞–ª–æ–≤–∞—Ç—å—Å—è –Ω–∞ –æ—Ç—Å—É—Ç—Å—Ç–≤–∏–µ –∞—Ä–≥—É–º–µ–Ω—Ç–æ–≤.
–û–¥–Ω–∞–∫–æ –º–æ–∂–Ω–æ –ø–µ—Ä–µ–¥–∞–≤–∞—Ç—å –∞—Ä–≥—É–º–µ–Ω—Ç—ã —Å –ø–æ–¥—Å—Ç–∞–Ω–æ–≤–æ—á–Ω—ã–º–∏ –∑–Ω–∞–∫–∞–º–∏.
–ù–∞–ø—Ä–∏–º–µ—Ä,

##### –°–µ–º–∞–Ω—Ç–∏–∫–∞ –≤—Å—Ç—Ä–æ–µ–Ω–Ω—ã—Ö –º–µ—Ç–æ–¥–æ–≤

–ü—Ä–∏–º–µ—Ä –º–µ—Ç–æ–¥–∞ `logged` –∏—Å–ø–æ–ª—å–∑—É–µ—Ç —Ç—Ä–∏ —Ä–∞–∑–Ω—ã—Ö —Ç–∏–ø–∞ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤, –∏–ª–ª—é—Å—Ç—Ä–∏—Ä—É—è, –∫–∞–∫ –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–∏–µ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞–µ—Ç —ç—Ç–∏ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã:

1. **–ü–∞—Ä–∞–º–µ—Ç—Ä—ã –ø–æ –∑–Ω–∞—á–µ–Ω–∏—é**. –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä —Å–æ–∑–¥–∞–µ—Ç `val` –ø—Ä–∏–≤—è–∑–∫—É –¥–ª—è –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –ø–æ –∑–Ω–∞—á–µ–Ω–∏—é. 
–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –∞—Ä–≥—É–º–µ–Ω—Ç–∞ –æ—Ü–µ–Ω–∏–≤–∞–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω —Ä–∞–∑ –ø–µ—Ä–µ–¥ —Å–æ–∫—Ä–∞—â–µ–Ω–∏–µ–º —Ç–µ–ª–∞ –º–µ—Ç–æ–¥–∞.
–≠—Ç–æ –≤–∏–¥–Ω–æ –ø–æ –ø–∞—Ä–∞–º–µ—Ç—Ä—É `level` –∏–∑ –ø—Ä–∏–º–µ—Ä–∞. 
–í –Ω–µ–∫–æ—Ç–æ—Ä—ã—Ö —Å–ª—É—á–∞—è—Ö, –∫–æ–≥–¥–∞ –∞—Ä–≥—É–º–µ–Ω—Ç—ã —è–≤–ª—è—é—Ç—Å—è —á–∏—Å—Ç—ã–º–∏ –ø–æ—Å—Ç–æ—è–Ω–Ω—ã–º–∏ –∑–Ω–∞—á–µ–Ω–∏—è–º–∏, 
–ø—Ä–∏–≤—è–∑–∫–∞ –æ–ø—É—Å–∫–∞–µ—Ç—Å—è –∏ –∑–Ω–∞—á–µ–Ω–∏–µ –≤—Å—Ç—Ä–∞–∏–≤–∞–µ—Ç—Å—è –Ω–∞–ø—Ä—è–º—É—é.
2. **–ü–∞—Ä–∞–º–µ—Ç—Ä—ã –ø–æ –∏–º–µ–Ω–∏**. –ö–æ–º–ø–∏–ª—è—Ç–æ—Ä —Å–æ–∑–¥–∞–µ—Ç `def` –ø—Ä–∏–≤—è–∑–∫—É –¥–ª—è –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –ø–æ –∏–º–µ–Ω–∏. 
–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –∞—Ä–≥—É–º–µ–Ω—Ç–∞ –æ—Ü–µ–Ω–∏–≤–∞–µ—Ç—Å—è –∫–∞–∂–¥—ã–π —Ä–∞–∑, –∫–æ–≥–¥–∞ –æ–Ω–æ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è, –Ω–æ –∫–æ–¥ —è–≤–ª—è–µ—Ç—Å—è –æ–±—â–∏–º.
–≠—Ç–æ –≤–∏–¥–Ω–æ –ø–æ –º–µ—Ç–æ–¥—É `message` –∏–∑ –ø—Ä–∏–º–µ—Ä–∞.
3. **–í—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã**. –í—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –Ω–µ —Å–æ–∑–¥–∞—é—Ç –ø—Ä–∏–≤—è–∑–æ–∫ –∏ –ø—Ä–æ—Å—Ç–æ –≤—Å—Ç—Ä–∞–∏–≤–∞—é—Ç—Å—è. 
–¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º, –∏—Ö –∫–æ–¥ –¥—É–±–ª–∏—Ä—É–µ—Ç—Å—è –≤–µ–∑–¥–µ, –≥–¥–µ –æ–Ω–∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç—Å—è.
–≠—Ç–æ –≤–∏–¥–Ω–æ –ø–æ –ø–∞—Ä–∞–º–µ—Ç—Ä—É `op` –∏–∑ –ø—Ä–∏–º–µ—Ä–∞.

–°–ø–æ—Å–æ–± –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç, —á—Ç–æ –≤—Å—Ç—Ä–∞–∏–≤–∞–Ω–∏–µ –≤—ã–∑–æ–≤–∞ –Ω–µ –∏–∑–º–µ–Ω–∏—Ç –µ–≥–æ —Å–µ–º–∞–Ω—Ç–∏–∫—É. 
–≠—Ç–æ –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –ø–µ—Ä–≤–æ–Ω–∞—á–∞–ª—å–Ω–∞—è –æ–±—Ä–∞–±–æ—Ç–∫–∞ (—Ä–∞–∑—Ä–µ—à–µ–Ω–∏–µ –ø–µ—Ä–µ–≥—Ä—É–∑–∫–∏, –Ω–µ—è–≤–Ω—ã–π –ø–æ–∏—Å–∫ –∏ —Ç. –¥.), 
–≤—ã–ø–æ–ª–Ω—è–µ–º–∞—è –ø—Ä–∏ –≤–≤–æ–¥–µ —Ç–µ–ª–∞ –≤—Å—Ç—Ä–æ–µ–Ω–Ω–æ–≥–æ –º–µ—Ç–æ–¥–∞, –Ω–µ –∏–∑–º–µ–Ω–∏—Ç—Å—è –ø—Ä–∏ –≤—Å—Ç—Ä–æ–µ–Ω–Ω–æ–º –º–µ—Ç–æ–¥–µ.

–ù–∞–ø—Ä–∏–º–µ—Ä, —Ä–∞—Å—Å–º–æ—Ç—Ä–∏–º —Å–ª–µ–¥—É—é—â–∏–π –∫–æ–¥:

```scala mdoc:silent

  def log(x: Any): Unit = println(x)

class RefinedLogger extends Logger:
  override def log(x: Any): Unit = println("Any: " + x)
  def log(x: String): Unit = println("String: " + x)

inline def logged[T](logger: Logger, x: T): Unit =
  logger.log(x)
```

–û—Ç–¥–µ–ª—å–Ω–∞—è –ø—Ä–æ–≤–µ—Ä–∫–∞ —Ç–∏–ø–∞ `logger.log(x)` —Ä–∞–∑—Ä–µ—à–∞–µ—Ç –≤—ã–∑–æ–≤ –º–µ—Ç–æ–¥–∞ `Logger.log`, –∫–æ—Ç–æ—Ä—ã–π –ø—Ä–∏–Ω–∏–º–∞–µ—Ç –∞—Ä–≥—É–º–µ–Ω—Ç —Ç–∏–ø–∞ `Any`. 

–¢–µ–ø–µ—Ä—å, —É—á–∏—Ç—ã–≤–∞—è —Å–ª–µ–¥—É—é—â–∏–π –∫–æ–¥:

```scala mdoc

```

–û–Ω —Ä–∞—Å—à–∏—Ä—è–µ—Ç—Å—è –¥–æ:

```scala mdoc

logger.log(x)
```
